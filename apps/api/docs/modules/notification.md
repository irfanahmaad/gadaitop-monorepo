# Notification Module

## Overview

The Notification module handles in-app notifications for users. Notifications are generated by system events and displayed in user dashboards.

## API Endpoints

| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| GET | `/v1/notifications` | List user notifications | Yes |
| GET | `/v1/notifications/unread-count` | Get unread count | Yes |
| GET | `/v1/notifications/:id` | Get notification | Yes |
| PATCH | `/v1/notifications/:id/read` | Mark as read | Yes |
| POST | `/v1/notifications/read-all` | Mark all as read | Yes |

## Features

- User-scoped notifications
- Read/unread tracking
- Unread count badge support
